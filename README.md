## 🚀 How to Run

Each Java file is self-contained with a `main` method. Compile and run individually:

```bash
# Compile
javac Arrays/BinarySearch.java

# Run
java -cp Arrays BinarySearch
```

> **Tip:** Use an IDE like IntelliJ IDEA or VS Code with the Java Extension Pack for the best experience.
